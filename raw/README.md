# Raw — Fuentes crudas (inmutables)

El usuario deposita aquí los **documentos fuente**: artículos, papers, imágenes,
datos, transcripciones, notas. Son **inmutables**: Hermes los LEE pero NUNCA los
modifica. Esta es la fuente de verdad del wiki.

## Convenciones
- Nombra los archivos de forma legible y estable (p.ej. `2026-leyes-mexico.md`).
- Si son muchos, agrúpalos en subcarpetas (`raw/2026/`, `raw/mexico/`…).
- Imágenes en `raw/assets/`.
- **No editar, no borrar** lo que el usuario ponga aquí.

## Fuente del método
- `raw/llm-wiki.md` — el patrón *LLM Wiki* que gobierna cómo Hermes mantiene
  este repo (ingest / query / lint, index.md, log.md).
