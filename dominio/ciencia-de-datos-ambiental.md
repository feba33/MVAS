# Ciencia de datos ambiental

La **ciencia de datos ambiental** aplica métodos de datos (estadística, ML, SIG) a
problemas de medio ambiente y clima. Conecta observación de la Tierra con decisiones de
sostenibilidad.

## Fuentes de datos
- **Teledetección (remote sensing):** satélites (Landsat, Sentinel, MODIS) y sensores
  in-situ para cobertura, vegetación, temperatura, calidad de aire/agua.
- **Modelado climático:** simulaciones de circulación global/regional (IPCC, CMIP).
- **SIG y series temporales:** monitoreo de deforestación, inundaciones, sequías.

## Métodos
- Limpieza y fusión de datos heterogéneos; *data cubes*; Python/R, GDAL, TensorFlow/PyTorch
  para detección y predicción.
- Analítica de sostenibilidad: huella de carbono, biodiversidad, riesgo físico.

## Aplicaciones
- Alerta temprana de desastres, agricultura de precisión, cumplimiento ESG, planeación
  urbana y conservación. Cruza con ciencia ciudadana y políticas de adaptación.
