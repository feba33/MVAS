# Ciencia de datos de salud (Health Data Science)

La **ciencia de datos de salud** aplica estadística, informática y aprendizaje automático a datos clínicos y biomédicos para mejorar la atención, la operación y la investigación. Es la unión de **health informatics** (gestión de información médica) con ciencia de datos moderna (modelos predictivos, ML).

## Fuentes y tipos de datos
- **EHR/EMR** (registros electrónicos), labs, imagen, genómica, dispositivos/wearables, claims, registros de salud pública.
- Calidad de los datos: completitud, sesgo, interoperabilidad (HL7/FHIR, SNOMED, LOINC, ICD-10).

## Tareas y métodos
- Modelos predictivos: riesgo de readmisión, sepsis, deterioro, estratificación de población.
- NLP sobre notas clínicas; cohortes para investigación; ensayos adaptativos.
- Operación: optimización de capacidad, no-shows, cadena de suministro farmacéutico.
- IA generativa para codificación, resumen, decision support (con supervisión clínica).

## Gobernanza y riesgos
- **Privacidad**: HIPAA (EE.UU.), GDPR/ datos especiales (UE), LFPDPPP (México), PHIPA (Canadá provincial).
- **Sesgo y equidad**: modelos entrenados en poblaciones no representativas.
- **Validación clínica**, explicabilidad, responsabilidad (vs model risk).
