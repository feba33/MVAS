# Protección de datos y privacidad (disciplina transversal)

## 📝 Conocimiento

### Marco
Disciplina que regula el tratamiento de datos personales. Los regímenes clave:
- **UE/RGPD** (2016/679): principios (licitud, minimización, finalidad), base legal, derechos ARCO/DSAR, transferencias internacionales, multas hasta 4% facturación o €20M.
- **EE.UU.:** CCPA/CPRA (California) y leyes estatales crecientes; enfoque sectorial federal (HIPAA, GLBA, COPPA).
- **Brasil:** LGPD (Ley 13.709/2018).
- **México:** LFPDPPP (reforma 2025); **Canadá:** PIPEDA; **Argentina/Colombia/Chile/Perú/Uruguay/Ecuador/Portugal/España:** leyes locales (ver nodos [sustrato](../../sustrato/)).
- **Asia:** PIPL (China), PDPA (Singapur/India).

### Conceptos transversales
DPO/CIPO, *privacy by design*, evaluaciones de impacto (DPIA), transferencias internacionales (cláusulas modelo, adecuación), brechas (notificación), gobernanza de datos (véase [gobierno-de-datos](../gobierno-de-datos/)).

### Relevancia para el usuario
- Cumplimiento obligatorio para cualquier negocio que trate datos de personas; afecta producto, marketing ([marketing](../marketing/)), RRHH ([recursos-humanos](../recursos-humanos/)) y tecnología ([computacion](../computacion/)).
- Conecta con [Chief Privacy Officer](../chief-privacy-officer/) y [gobierno-de-datos](../gobierno-de-datos/).
