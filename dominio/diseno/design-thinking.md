# Design Thinking (y UX Research)

## 📜 Cita
> "Design thinking refers to the set of cognitive, strategic and practical procedures used by designers ... and to the body of knowledge ... about how people reason when engaging with design problems." — Wikipedia (en)

## 📝 Conocimiento
- Procedimientos cognitivos/estratégicos/prácticos del diseño; usado para innovar productos y servicios.
- **UX research:** observación, entrevistas, tests de usabilidad — centro del enfoque centrado en el usuario.
- Doble diamante: divergir (explorar) → convergir (definir) → prototipar → testear.
- **Relevancia MVAS:** base del dominio `diseno`; alimenta `rol/ux-ui-lead`.
