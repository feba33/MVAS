# Plantas industriales

## Definición
Disciplina de ingeniería que abarca el diseño, instalación, operación y mejora de
instalaciones de producción (plantas de proceso, fábricas, generación). Integra
procesos, utilidades, layout y mantenimiento.

## Componentes
- **Proceso:** reactores, intercambiadores, líneas de ensamble, instrumentación.
- **Utilidades (utilities):** vapor, aire comprimido, agua, energía, refrigeración.
- **Layout:** flujo de materiales, distancias, seguridad, expansión.
- **Mantenimiento:** preventivo/predictivo, confiabilidad de activos.

## Ciclo de vida
- Ingeniería conceptual → FEED → detalle → procura → construcción → comisionamiento
  → operación → retrofitting.
- Gestión de capital (CAPEX) y OPEX; KPIs: OEE, disponibilidad, yield.

## Estándares y seguridad
- Normas API/ASME/ISA; integración con `dominio/ciberseguridad-ot` (ICS/SCADA).
- Seguridad de procesos (OSHA PSM, Seveso).
