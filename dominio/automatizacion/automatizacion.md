# Automatización / RPA / hyperautomation / IA agentes

## 📜 Cita
> "Automation is the technology by which a process or procedure is performed with minimal human assistance." — Wikipedia (Automation)

## 📝 Conocimiento
- **Definición:** uso de tecnología (mecánica, electrónica, software) para ejecutar tareas/procesos con mínima intervención humana. En negocios abarca desde la fábrica hasta el back-office.
- **Modalidades:**
  - **Automatización industrial / control:** PLC, SCADA, IIoT (ver `dominio/iot/`, `dominio/manufactura/`).
  - **BPA (Business Process Automation):** orquestación de flujos de trabajo de back-office.
  - **RPA (Robotic Process Automation):** "bots" de software que ejecutan reglas sobre interfaces existentes (sin API) — rápido de desplegar, frágil a cambios de UI.
  - **IA / automatización inteligente:** ML/NLP para decisiones no deterministas; **IA agentes** (agentic AI) que planifican y ejecutan tareas multi-paso (ver `dominio/ia-generativa/`).
  - **Low-code / no-code:** plataformas que democratizan la construcción de automatizaciones.
- **Hiperautomatización (Gartner):** orquestar RPA + process mining + IA + case management + APIs para automatizar extremo a extremo; incluye descubrimiento de procesos y gobernanza.
- **Beneficios:** eficiencia, calidad/consistencia, reducción de costo, cumplimiento auditable, escalabilidad.
- **Retos:** integración con sistemas legacy, gestión del cambio/rediseño de procesos, gobernanza y riesgo de la IA (sesgo, alucinaciones), impacto laboral (reskilling).
