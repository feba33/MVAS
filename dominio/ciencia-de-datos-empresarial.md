# Ciencia de datos empresarial

La **ciencia de datos empresarial** aplica el campo interdisciplinario de la
ciencia de datos (matemáticas, estadística, computación, método científico)
a problemas de negocio: obtener, tratar, analizar y presentar información
para decisiones y productos.

## Práctica empresarial
- **Ciclo de vida:** adquisición → limpieza → exploración → modelado
  (estadístico/ML) → despliegue → monitoreo.
- **MLOps:** operacionalización de modelos, CI/CD de ML, feature store,
  monitoreo de deriva (ver `dominio/gestion-de-modelos`).
- **Embed en producto:** recomendaciones, scoring, pronósticos, pricing
  dinámico.
- **Gobierno:** linaje, calidad y `rol/head-of-data-governance`.

## Diferencia con analítica
La ciencia de datos construye modelos predictivos; la analítica de negocio
(BI) describe y monitorea lo ocurrido. Se apoyan mutuamente.
