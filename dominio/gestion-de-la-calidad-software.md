# Gestión de la calidad de software

## Definición
La **SQA (Software Quality Assurance)** asegura que el producto de software cumple
requisitos funcionales y no funcionales. Abarca estrategia de pruebas, calidad de
código, gestión de defectos y métricas de calidad.

## Prácticas
- **Testing strategy:** unit/integration/e2e, automatización, shift-left (véase
  `dominio/computacion/testing-qa`).
- **Calidad de código:** revisiones, linters, análisis estático, deuda técnica.
- **Métricas:** densidad de defectos, escape rate, cobertura, MTTR.
- **SLA de calidad / fiabilidad** (véase `dominio/plataforma-sre` y observabilidad).
- **ISO 9001 / ISTQB** como marcos de referencia.

- Converge con `dominio/computacion/plataforma-sre`, `dominio/observabilidade`.
