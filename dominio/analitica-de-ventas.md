# Analítica de ventas (Sales Analytics)

La **analítica de ventas** es la disciplina de medir y predecir el desempeño comercial a partir de datos del CRM, pipeline y actividad. Su propósito es convertir el ruido de dashboards en decisiones: dónde hay cuellos de botella, si hay suficiente cobertura y qué pronóstico es realista.

## Métricas núcleo (las que importan)
- **Pipeline coverage**: ¿hay suficiente pipeline para el número? Benchmark 3–6x cuota; un equipo con 30% win rate necesita ~3.3x.
- **Win rate**: % de oportunidades que cierran (por etapa, segmento, rep).
- **Sales velocity**: (oportunidades × deal size × win rate) ÷ días = valor diario de cierre.
- **Sales cycle length** y **stage conversion** (razón entre etapas).
- **Deal aging**: deals estancados que probablemente no cierran.
- **Pipeline creation rate**: ¿entra pipeline nuevo a ritmo saludable?

## Práctica
- CRM (Salesforce, HubSpot) como fuente de verdad; limpieza de datos es prerequisito.
- Forecasting: weighed pipeline, cohortes, tendencias históricas.
- A/B en mensaje/outreach; análisis de pérdidas (win/loss).
- Conexión con **RevOps**, **comp planning** y **sales strategy**.

## Relación con otros nodos
