# Ingeniería de Machine Learning (ML Engineering)

## 📜 Cita
> "ML engineering is the discipline of taking machine learning models from research notebooks into reliable, scalable production systems, owning the full lifecycle from data to serving." — conocimiento general

## 📝 Conocimiento
- Puente entre la ciencia de datos y la ingeniería de software: lleva modelos desde el notebook a **producción fiable y escalable**.
- **Entrenamiento:** selección de datos, feature engineering, ajuste de hiperparámetros y validación; suele requerir cómputo distribuido/GPU.
- **Serving (inferencia):** exponer el modelo mediante APIs/batch/streaming, con latencia, throughput y versionado controlados.
- **MLOps:** prácticas (CI/CD para modelos, monitorización de drift, registro de modelos y reproducibilidad) que automatizan el ciclo de vida del ML.
- **Feature store:** repositorio centralizado de features para entrenamiento y servicio, garantizando consistencia online/offline y reutilización.
- El desafío clave no es el modelo sino el **sistema** (datos, pipelines, despliegue, monitoreo) que lo rodea.
