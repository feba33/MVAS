# Ingeniería de Datos

## 📜 Cita
> "Data engineering is the practice of designing and building systems for collecting, storing, and analyzing data at scale, so that los datos estén disponibles y sean fiables para consumidores downstream como analistas y modelos de ML." — conocimiento general

## 📝 Conocimiento
- Disciplina que diseña y construye sistemas para **recolectar, almacenar y analizar datos a escala** de forma fiable.
- **Pipelines:** flujos que mueven y transforman datos desde fuentes hasta destinos (batch o streaming), con control de calidad y observabilidad.
- **ETL vs ELT:** en ETL se transforma antes de cargar; en ELT se carga primero (p. ej. en un data lake/warehouse) y se transforma después aprovechando el motor de almacenamiento.
- **Data warehouses / data lakes:** almacenes columnar estructurados (warehouse) o repositorios de datos en crudo (lake) que centralizan la analítica.
- **dbt:** herramienta de transformación *in-warehouse* basada en SQL y modelos versionados (la "T" de ELT, hecha repositorio).
- **Spark:** motor de procesamiento distribuido para grandes volúmenes (batch y streaming) sobre clústeres.
