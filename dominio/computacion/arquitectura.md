# Arquitectura de Software

## 📜 Cita
> "Software architecture is the set of structures needed to reason about a software system ... Everything is a trade-off. 'Why is more important than how'." — Wikipedia (en)

## 📝 Conocimiento
- Conjunto de estructuras (elementos, relaciones, propiedades) para razonar el sistema; metáfora del plano del edificio.
- **Dos leyes:** "todo es trade-off"; "el porqué importa más que el cómo".
- **Modelo C4** para documentar; las características arquitectónicas (NFR) guían el diseño.
- Decisiones costosas de cambiar una vez implementadas → importa el "architectural kata".
