# Paradigmas de Programación

## 📜 Cita
> "A programming paradigm is a relatively high-level way to conceptualize and structure the implementation of a computer program. A programming language can be classified as supporting one or many paradigms." — Wikipedia (en)

## 📝 Conocimiento
- **Imperativo** (controla flujo y estado): procedural; **orientado a objetos** (objetos con estado+comportamiento; class-based vs prototype-based); object-based.
- **Declarativo** (describe el qué, no el cómo): **funcional**, lógico, reactivo.
- Un lenguaje puede soportar varios paradigmas (p.ej. Python, Scala, Kotlin multiparadigma).
- Otras dimensiones: modelo de ejecución (efectos secundarios), organización del código, sintaxis.
