# Ciencia de datos (Data Science)

## 📝 Conocimiento
- **Definición:** campo interdisciplinario que extrae conocimiento e insight de datos mediante métodos científicos, estadística, programación y aprendizaje automático.
- **Ciclo CRISP-DM:** entendimiento de negocio → entendimiento de datos → preparación (limpieza/wrangling) → modelado → evaluación → despliegue; el estándar de facto para proyectos de DS/ML.
- **Modelado:** regresión, clasificación, clustering, series de tiempo, NLP y deep learning; validación cruzada, métricas (accuracy, AUC, RMSE, F1), ajuste de hiperparámetros.
- **Herramientas:** Python (pandas, scikit-learn, PyTorch) y R; SQL; visualización (matplotlib, Tableau); orquestación con `dominio/computacion` (notebooks, pipelines).
- **vs** `dominio/estadistica` (fundamentos teóricos), `dominio/analitica-negocio` (BI/decisiones) y `dominio/analitica-de-mercado`; rol análogo en `rol/data-scientist`, `rol/data-analyst`, `rol/ml-engineer`.
- **Desafíos:** calidad de datos, sesgo, reproducibilidad, puente modelo→valor (`dominio/inteligencia-competitiva`).
