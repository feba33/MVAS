# Procurement Manager / Jefe de compras

## 📝 Conocimiento
- **Definición:** el procurement manager (jefe de compras/abastecimiento) lidera la adquisición de bienes y servicios externos: estrategia de sourcing, negociación, contratos y gestión de proveedores (proceso P2P).
- **Proceso:** identificar necesidad → especificar → RFx/licitación → evaluar → adjudicar → contratar → gestionar proveedor (7 pasos típicos). Busca mejor precio/calidad/plazo minimizando riesgo (fraude, colusión).
- **Alcance:** público (government procurement) y privado; incorpora criterios ESG/sostenibilidad y resiliencia de cadena.
- **Diferenciación:** vs `rol/chief-procurement-officer` (estrategia corporativa), vs `rol/supply-chain-manager` (plan/demanda/logística), y vs `dominio/compras` / `dominio/cadena-suministro`.
- **Relevancia:** conecta con `dominio/compras`, `dominio/cadena-suministro`, `rol/chief-procurement-officer`, `rol/supply-chain-manager` y `dominio/consultoria`.
