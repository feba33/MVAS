# Event Manager

## 📝 Conocimiento
- **Qué hace:** planea, produce y gestiona **eventos** (ferias, conferencias, congresos, eventos corporativos, incentivos, lanzamientos) — presupuesto, logística, venue, proveedores, agenda, registro, experiencia del asistente y medición de ROI/leads.
- **Segmento MICE:** Meetings, Incentives, Conferences, Exhibitions; incluye híbrido/virtual (webinars, streams). Industria de agencias y in-house (marketing, PR, RRII).
- **Habilidades:** gestión de proyectos, negociación con proveedores, gestión de riesgos (seguridad, contingencias), analítica de asistencia y conversión.
- **Diferenciación:** produce el evento vs *Marketing Manager* (estrategia de marca) y *Head of Marketing Ops* (plataforma/campañas).
