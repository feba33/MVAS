# Security Analyst

## 📝 Conocimiento
- **Definición:** analista de ciberseguridad que monitoriza, detecta y triage amenazas en un SOC (Security Operations Center) o equipo de respuesta.
- **Responsabilidades:** monitorización de SIEM/EDR, triaje de alertas (tier 1-2), análisis de vulnerabilidades, investigación de incidentes, colaboración con equipos de respuesta (IR) y threat intelligence.
- **Habilidades:** redes, sistemas operativos, herramientas de detección (SIEM, IDS/IPS), pensamiento analítico, certificaciones (CompTIA Security+, CEH, GCIH).
- **vs** `rol/penetration-tester` (ofensivo/ethical hacking), `rol/security-engineer` (implementa controles), `rol/security-architect` (disseña), `rol/ciso` (ejecutivo).
- **Cómo un LLM opera aquí:** resúmenes de alertas, borradores de informes de incidentes, guías de triaje, explicación de TTPs.
