# IT Manager

## 📝 Conocimiento
- **Función:** opera la **TI interna** — infraestructura (red, servidores, endpoints, nube híbrida), soporte/help desk, gestión de proveedores de tecnología, presupuesto de TI y continuidad de servicios.
- **Entrega de servicios:** SLA, mesa de ayuda, gestión de cambios, inventario de activos (ver `dominio/gestion-de-activos`).
- **vs roles vecinos:** `rol/head-of-it` (ejecutivo, estrategia de TI alineada al negocio), `rol/ciso` (seguridad), `rol/devops-engineer` (plataforma/desarrollo); el IT Manager es la capa operativa.
- **Habilidades:** gestión de incidentes, proveedores, presupuesto, liderazgo de equipos de soporte.
- **Marco:** ITIL, COBIT (ver `dominio/gobierno-de-ti`).
