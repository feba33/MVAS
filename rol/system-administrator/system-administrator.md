# System Administrator (SysAdmin)

## 📝 Conocimiento
- **Rol:** mantiene la **infraestructura TI** (servidores, sistemas operativos, redes, almacenamiento, backups y seguridad básica); garantiza disponibilidad y estabilidad de los sistemas.
- **Tareas:** *provisioning*, parches, monitoreo, automatización (scripts bash/Python), gestión de usuarios/permisos, planes de recuperación (DR) y soporte nivel 2/3.
- **Habilidades:** Linux/Windows, scripting, redes, virtualización/cloud y seguridad operativa.
- **Certificaciones:** CompTIA, RHCSA, MCSA, AWS/Azure.
