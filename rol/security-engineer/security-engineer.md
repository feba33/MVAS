# Security Engineer (Ingeniero de Seguridad)

## 📝 Conocimiento

### Definición
El **Security Engineer** diseña e implementa controles de seguridad para sistemas, infraestructura y aplicaciones; protege confidencialidad, integridad y disponibilidad.

### Responsabilidades
- **AppSec:** SDLC seguro, SAST/DAST, gestión de vulnerabilidades.
- **Cloud security:** IAM, segmentación de red, cifrado, hardening.
- **Threat modeling** (STRIDE), respuesta a incidentes, automatización de seguridad.
- Colabora con ingeniería de producto y plataforma.

### Diferenciación
- vs `rol/ciso`: el CISO fija estrategia/gobernanza; el engineer la ejecuta técnicamente.
- vs analista de seguridad: el engineer construye defensas; el analista monitorea/responde.
- Conexión con `dominio/ciberseguridad-gobierno` (ISO 27001) y `dominio/computacion/ciberseguridad`.

### Relevancia
Esencial ante amenazas crecientes; habilitador de compliance (SOC 2, ISO 27001).
