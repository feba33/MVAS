# Project Manager (Gestor de Proyectos)

## 📝 Conocimiento

### Definición
El **Project Manager (PM)** conecta la **estrategia con la ejecución**: traduce metas en planes
accionables, alinea equipos cross-funcionales y entrega **a tiempo, dentro del presupuesto y a
especificación**. Actúa como enlace entre stakeholders y el equipo de proyecto.

### Cinco responsabilidades centrales
1. **Planear el alcance** (WBS, cronograma, recursos, presupuesto).
2. **Liderar equipos** (dirección, motivación, gestión de conflictos).
3. **Seguir el progreso** (KPIs, hitos, reporting).
4. **Comunicar con stakeholders** (expectativas, estatus).
5. **Gestionar riesgos** (identificación, mitigación).

### Ciclo de vida del proyecto
Iniciación → Planificación → Ejecución → Monitoreo y Control → Cierre (charter, WBS, hitos,
comunicación). Marco **PMBOK** del PMI; certificación **PMP**.

### Metodologías
**Agile/Scrum** (iterativo, sprints, product owner) para entornos de cambio rápido; waterfall
para proyectos predecibles. La IA está desplazando el rol de coordinación administrativa hacia
**impacto estratégico** (monday.com, 2026).
