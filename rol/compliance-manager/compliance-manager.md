# Compliance Manager

## 📝 Conocimiento
- **Rol:** ejecuta el **programa de cumplimiento** día a día — monitoreo y testing de controles, debida diligencia (AML/KYC), reportes regulatorios, capacitación, remediación de hallazazgos y gestión de políticas.
- **Frameworks:** ISO 37301 (Sistema de Gestión de Cumplimiento certificable), COSO, modelo de **tres líneas de defensa**; apoya a `rol/chief-compliance-officer` y a `rol/compliance-officer`.
- **Ámbitos:** anticorrupción (FCPA/UK Brias), PLD/FT (ver `sustrato/mexico/pld-uif`), protección de datos (`dominio/proteccion-de-datos`), conducta de mercado.
- **vs** `rol/compliance-officer` (frecuentemente el mismo; "officer" implica responsabilidad senior/certificación) y `rol/chief-compliance-officer` (ejecutivo, dueño del programa). El manager es el nivel operativo/táctico.
- **Relevancia:** conecta con `dominio/cumplimiento-regulacion`, `dominio/control-interno`, `rol/regulatory-affairs`, `rol/internal-auditor`.
