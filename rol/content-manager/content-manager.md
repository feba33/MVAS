# Content Manager

## 📝 Conocimiento
- **Función:** define la **estrategia de contenido**, arma el calendario editorial, coordina producción (redacción/edición/diseño), publica y distribuye (web, blog, social, email) y mide engagement/SEO.
- **Opera el ciclo:** ideación → brief → creación → aprobación → publicación → promoción → análisis (ver `dominio/marketing`, `dominio/analitica-marketing`).
- **vs roles vecinos:** `rol/chief-content-officer` (visión ejecutiva/narrativa de marca), `rol/marketing-manager` (dueño de marketing), `rol/social-media-manager` (canales sociales), `rol/community-manager` (comunidad).
- **Habilidades:** storytelling, SEO, gestión de CMS, edición, data-ligera.
- **Relevancia:** contenido como motor de demanda (ver `dominio/demand-generation`, `dominio/inbound`).
