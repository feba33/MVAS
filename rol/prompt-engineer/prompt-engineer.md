# Prompt Engineer (ingeniería de prompts para LLMs)

## 📝 Conocimiento
- **Rol:** diseña y refina instrucciones (prompts) dadas a modelos generativos de IA para obtener salidas más precisas, relevantes o útiles.
- **Técnicas:** few-shot / multi-shot, chain-of-thought, tree-of-thought, asignación de rol (role assignment); context engineering (gestión de system instructions, metadata, tools, tokens).
- **Ciclo de trabajo:** definir tarea, redactar prompt, evaluar salida (eval de modelos), iterar; incluye RAG para mayor alcance/precisión.
- **Riesgos:** prompt injection (ataque de ciberseguridad vía prompts maliciosos); necesidad de guardrails y evaluación robusta (ver `dominio/seguridad-ia`).
- **Tendencia:** surgió en el auge de IA 2020s como "business capability"; el título propio perdió tracción ante modelos que auto-generan mejores prompts y training general de empleados en prompting — evoluciona hacia "AI/LLM ops" y evaluación.
- **Relevancia:** conecta con `rol/ml-engineer`, `rol/data-scientist`, `dominio/ia-generativa`, `dominio/llm` y `rol/ai-product-manager`.
