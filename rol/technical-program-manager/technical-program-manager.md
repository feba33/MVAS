# Technical Program Manager (TPM)

## 📝 Conocimiento

### Misión
- Coordina **programas técnicos** (conjunto de proyectos de múltiples equipos) para lograr entregas a tiempo: gestiona **dependencias**, **riesgo**, alcance y comunicación a ejecutivos. Puente entre ingeniería, producto y negocio.

### Diferencia con PM / Program Manager
- El **TPM** tiene profundidad técnica (puede leer diseños/arquitectura) pero no necesariamente escribe código; el **PM** se enfoca en producto/mercado y el **Program Manager** en portafolios más amplios. El TPM brilla en entregas multi-equipo complejas (migraciones, plataformas, launches).

### Prácticas
- Roadmaps, RACI, gestión de incidencias/riesgos, dashboards de entrega, reuniones de sincronización.
