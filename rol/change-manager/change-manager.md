# Change Manager

## 📝 Conocimiento
- **Rol:** planifica y ejecuta la **gestión del cambio organizacional** para que iniciativas (transformación, nuevos sistemas, reestructuras) sean adoptadas y sostenidas por las personas.
- **Modelos:** **ADKAR/Prosci** (Awareness, Desire, Knowledge, Ability, Reinforcement), **Kotter** (8 pasos), **Lewin** (unfreeze-move-refreeze), *Change Curve*.
- **Prácticas:** evaluación de impacto, mapeo de *stakeholders*, estrategia de comunicación, planes de capacitación, *coaching* a líderes, medición de adopción y resistencia.
- **Entregables:** plan de cambio, *readiness assessment*, comunicaciones, *training plan*, *reinforcement* para evitar reversión.
- **Diferenciación:** vs `rol/chief-transformation-officer` (el CTO lidera la transformación a nivel ejecutivo/estratégico; el Change Manager opera el cambio a nivel de iniciativa/proyecto). Conecta con `rol/chief-learning-officer`.
- **Relevancia:** habilita la ejecución de estrategia. Conecta con `dominio/gestion-cambio`, `rol/chief-transformation-officer`, `rol/chief-learning-officer` y `dominio/operaciones`.
