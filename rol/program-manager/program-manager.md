# Program Manager (program management, PgMP, PMO)

## 📝 Conocimiento
- **Definición:** el *program management* supervisa un grupo de proyectos alineados con la estrategia, metas y misión de la organización, para mejorar su desempeño. Es **distinto de *project management*** (que gestiona un proyecto individual con entregables).
- **Jerarquía de gestión:** (1) **proyecto** → entregables; (2) **programa** → beneficios/resultados agregados; (3) **portafolio** → valor estratégico. El Program Manager gobierna el nivel 2.
- **Marcos y certificaciones:** *Standard for Program Management* del **PMI** y credencial **PgMP**; integra la gestión de portafolio (PfMP). Apoya/habita la **PMO** (Project/Program Management Office).
- **Responsabilidades:** gobernanza de múltiples proyectos, **gestión de beneficios**, mapa de dependencias, *stakeholders* (patrocinadores, negocio, TI), *roadmap* y alineación estratégica, y gestión del cambio organizacional.
- **Habilidades:** liderazgo, comunicación, visión financiera y facilitación; menos ejecución técnica, más orquestación.
