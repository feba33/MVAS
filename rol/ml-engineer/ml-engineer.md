# ML Engineer (Ingeniero de Machine Learning)

## 📝 Conocimiento

### Definición
El **ML Engineer** construye, entrena, despliega y mantiene modelos de aprendizaje automático en producción; une la ciencia de datos y la ingeniería de software.

### Responsabilidades
- Pipelines de features, entrenamiento y servicio (model serving); **MLOps** (CI/CD de modelos, monitoreo de drift, reentrenamiento).
- Escalado en la nube (SageMaker, Vertex AI, Azure ML), contenedores/Kubernetes, feature stores.
- Calidad: evaluación, explicabilidad, mitigación de sesgo.

### Diferenciación
- vs `rol/data-scientist`: el científico experimenta/descubre; el ML engineer lleva a producción.
- vs `rol/data-engineer`: el engineer de datos construye pipelines de datos; el ML engineer consume esos datos para modelos.
- Reporta a Head of ML / Engineering; cerca de `rol/chief-ai-officer`.

### Relevancia
Clave para productizar IA (`dominio/ia-generativa`, `dominio/computacion/ml-engineering`); escasez de talento eleva su valor.
