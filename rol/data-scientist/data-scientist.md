# Data Scientist (Científico de datos)

## 📜 Cita
> "El científico de datos extrae conocimiento y valor de los datos mediante métodos
> científicos, procesos, algoritmos y sistemas." — Wikipedia

## 📝 Conocimiento
- **Rol:** analiza datos para resolver problemas de negocio y construir modelos
  predictivos; combina estadística, programación y conocimiento del dominio.
- **Responsabilidades:** definir la pregunta de negocio; adquirir y limpiar datos; EDA y
  visualización; feature engineering; modelado (regresión, clasificación, series de tiempo,
  NLP, ML); validación y métricas; despliegue y monitoreo de modelos; comunicar resultados a
  stakeholders.
- **Stack:** Python/R, SQL, bibliotecas (pandas, scikit-learn, PyTorch/TensorFlow), nube y
  data warehouses; colabora con Data Engineering (cross-ref `rol/data-engineer`).
- **DS vs Data Analyst:** el analista reporta métricas del pasado; el científico construye
  modelos y predice (cross-ref `dominio/estadistica`, `dominio/computacion/ml-engineering`).
- **DS vs Data Engineer:** el ingeniero construye pipelines/infra; el científico los consume
  para modelar.
- **Reporta a:** Head of Data / Analytics / CDO; en equipos de producto o analítica.
