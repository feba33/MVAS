# DevOps Engineer

## 📝 Responsabilidades
- Automatiza el ciclo de vida de software: **CI/CD** (integración/despliegue continuos), infraestructura como código (Terraform, CloudFormation, Pulumi), gestión de configuración y despliegues.
- Construye y mantiene *pipelines*, ambientes y observabilidad (métricas, *logs*, trazas).
- Colabora con desarrollo y operaciones para acortar *lead time*, aumentar frecuencia de despliegue y mejorar confiabilidad.

## 🔍 Diferenciación
- De [Platform Engineer](../../rol/platform-engineer/) (construye la plataforma interna/*golden paths* para otros equipos) y de [Security Engineer](../../rol/security-engineer/). DevOps es una cultura + rol que une Dev y Ops.

## 💡 Stack típico
Git, GitHub/GitLab CI, Jenkins, Docker/Kubernetes, Terraform, Prometheus/Grafana, nubes ([cloud-computing](../../dominio/cloud-computing/)).
