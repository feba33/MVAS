# Cloud Architect (Arquitecto de nube)

## 📝 Conocimiento

### Misión
- Diseña **arquitecturas cloud** (IaaS/PaaS/SaaS) en AWS, Azure o GCP; define estándares, patrones y la hoja de ruta de migración desde on-prem. Trabaja con stakeholders de negocio y equipos técnicos.

### Marco Well-Architected
- Pilares: **excelencia operativa, seguridad, fiabilidad, eficiencia de rendimiento, optimización de costos** (FinOps). Considera **multi-cloud/híbrido** para evitar lock-in y cumplir residencia de datos.
- Decisiones: redes/VPC, identidad (IAM), datos, serverless vs contenedores, DR/BC.

### Habilidades y vecinos
- Conocimiento profundo de proveedores, Terraform/IaC, seguridad y economía cloud.
